================================================================================
BUILD INSTRUCTIONS
================================================================================

STANDARD BUILD
  cd acraas/sdk
  mkdir build && cd build
  cmake ..
  make -j$(nproc)

WITH TESTS
  cmake -DBUILD_TESTS=ON ..
  make
  ctest --verbose

ANDROID BUILD
  export NDK=/path/to/android-ndk-r21
  cmake -DBUILD_ANDROID=ON \
    -DCMAKE_TOOLCHAIN_FILE=$NDK/build/cmake/android.toolchain.cmake \
    -DANDROID_PLATFORM=android-21 \
    -DANDROID_ABI=armeabi-v7a \
    ..
  make -j$(nproc)

INSTALLATION
  sudo make install

PERFORMANCE SPECIFICATIONS
================================================================================

CPU USAGE
  Target: < 2%
  Configurable via capture_interval_sec
  Background threads with sleep intervals

MEMORY USAGE
  SDK overhead: 2-5 MB
  Audio buffer: 0.2 MB
  Cache (500 fingerprints): ~50 MB max
  Total resident: 10-20 MB

NETWORK
  Batch transmission: 20 fingerprints/batch (configurable)
  Flush interval: 60 seconds (configurable)
  Backoff: 1s → 2s → 4s → 8s → ... → 300s
  Timeouts: 10s connect, 30s read

STORAGE
  Cache database: /var/lib/acr/fingerprint_cache.db
  Device ID: /var/lib/acr/device_id
  Consent: /var/lib/acr/consent
  Max cache: 500 fingerprints (~50 MB)
