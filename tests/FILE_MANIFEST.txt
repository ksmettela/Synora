USAGE QUICK START
=================

1. Install dependencies:
   pip install -r tests/requirements.txt

2. Run fast tests (82 tests in ~2 minutes):
   pytest tests/integration/ -m "not slow" -v

3. Run full suite (85 tests in ~10 minutes):
   pytest tests/integration/ -v

4. Run load test (100 concurrent users):
   locust -f tests/load/locustfile.py --headless -u 100 -r 10 --run-time 60s

5. Run specific test class:
   pytest tests/integration/test_ingestor.py::TestAuthentication -v

For more options, see README.md
