- Enriches events with household_id (SHA256 hash of device_ids grouped by /24 subnet)
- Partitions by watch_date, network, genre
- Writes to acraas.viewership Iceberg table with PARQUET format
- Target file size: 128MB
- Checkpoint every 60 seconds
- Exactly-once processing semantics

Files created:
1. build.sbt - SBT build configuration for Spark 3.5, Scala 2.12, Iceberg 1.4.0
2. src/main/scala/io/acraas/spark/HouseholdAggregationJob.scala - Daily aggregation job
3. src/main/scala/io/acraas/spark/RetentionCleanupJob.scala - Data retention enforcement
4. src/main/scala/io/acraas/spark/IcebergMaintenanceJob.scala - Iceberg table maintenance

Job details:

HouseholdAggregationJob:
- Reads viewership data from last 7 days
- Computes genre_affinity_scores (normalized 0-100 per genre)
- Computes daypart_patterns (morning/afternoon/primetime/latenight percentages)
- Computes brand_affinity_signals (placeholder with TODO for ad catalog join)
- Writes to acraas.household_aggregates Iceberg table

RetentionCleanupJob:
- Deletes device-level viewership data older than 90 days
- Archives household aggregates older than 365 days to S3 Glacier
- Logs deletion counts to deletion_audit table
- Verifies expected deletion ranges

IcebergMaintenanceJob:
- Expires snapshots older than 7 days (retains last 5)
- Removes orphan files older than 7 days
- Rewrites small data files with binpack strategy and zorder sort
