GETTING STARTED
===============
1. Copy .env.example to .env
2. Update configuration as needed
3. Start services: make docker-up
4. View logs: make docker-logs
5. Test ingestion: curl -X POST http://localhost:8080/v1/fingerprints ...
6. Check metrics: curl http://localhost:9090/metrics
7. Load test: ./scripts/load-test.sh 1000 10 50
8. Run tests: go test -v ./...

PRODUCTION DEPLOYMENT
=====================
1. Build image: make docker-build
2. Push to registry: docker push acraas/fingerprint-ingestor:latest
3. Deploy to Kubernetes or Docker Swarm
4. Configure environment variables
5. Mount GeoIP database volume
6. Monitor metrics at /metrics endpoint
